# Skann markedet — 20 designvarianter

Frittstående design-galleri for «Skann markedet»-siden. **Rører ikke produksjonskode** (`src/`).

## Åpne galleriet

Fra prosjektroten:

```bash
cd /Users/ben/nye-firma-plattform
npx serve designs/skann-markedet-gallery -p 3456
```

Åpne deretter: **http://localhost:3456**

Alternativt med Python:

```bash
python3 -m http.server 3456 --directory designs/skann-markedet-gallery
```

## Bruk

- **Oversikt:** 20 miniatyrmockups i et rutenett
- **Klikk** på en variant for fullskjermvisning
- **Noter variant-ID** (f.eks. `08-sidebar-filtre`) når du har valgt favoritt
- **Esc** eller «Lukk» for å gå tilbake
